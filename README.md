# ♟️ Python 五子棋双人对战 & AI 版
### Developed by Deng Weifeng et Shen Zhicheng

这是一个基于 Python `tkinter` 库开发的五子棋游戏。项目采用模块化设计，实现了**双人对战**、**人机对战（AI）**、**实时计分**、**悔棋**以及**棋盘重置**等完整功能。

## ✨ 功能特性

- **双人对战**：支持两名玩家在同一台设备上轮流落子。
- **人机对战 (AI)**：内置基于“评分法”的 AI 算法，电脑会自动评估棋局并落子。
- **实时计分**：自动记录黑棋与白棋的胜场数。
- **悔棋功能**：支持撤销上一步操作，防止手滑。
- **状态管理**：包含“重置棋盘”（保留比分）和“重置得分”功能。
- **胜负判定**：自动检测五子连珠，弹出胜利提示。

## 🛠️ 核心模块说明
1. **界面层 \(chessboard.py)**
BoardUI 类：负责绘制 8x8 (嗯这个是可以改的) 的棋盘网格和棋子。
事件处理：通过 canvas.bind("\<Button-1>", self.on_click) 监听鼠标点击，计算落子坐标。
状态栏：实时显示“轮到黑棋/白棋”以及当前比分。
2. **逻辑层 \(game_logic.py)**
GameLogic 类：维护一个二维数组 self.board 存储棋盘状态（0:空, 1:黑, 2:白）。
胜负算法：check_win 方法通过四个方向（水平、垂直、两条对角线）扫描，判断是否连成五子。
悔棋逻辑：使用 move_history 栈记录每一步坐标，悔棋时弹出栈顶并清空对应棋盘位置。
3. **AI 算法 \(ai.py)**
SimpleAI 类：实现基础的评分法 (Scoring Method)。
核心逻辑：
遍历棋盘所有空位。
模拟落子，计算该位置在八个方向上的棋型（连五、活四、活三等）。
根据预设权重累加分数，选择分数最高的位置落子。
